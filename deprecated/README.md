# Deprecated Files

These files are legacy versions of the Feedback Analysis Tool, kept for historical reference.

## Files

### `app_llama.py`
- Original Ollama-only version
- Basic feedback analysis without visualizations
- Superseded by `app_llama3v2.py`

### `llama3app_cfa.py`
- Customer Feedback Analysis variant
- Similar functionality to app_llama.py
- Superseded by `app_llama3v2.py`

## Current Version

The main application is now `app_llama3v2.py` in the root directory, which includes:
- Support for multiple LLM providers (Ollama, Groq)
- Row-level tagging
- Visual dashboard
- Executive summary generation
- Export functionality
- Trend analysis

## History

These files were part of the original project created in June 2024.
